Painel sobre Interferência

Nesse trabalho, eu e meu grupo (Erico, Gabriel, Gustavo, João Pedro e Sirlan) tivemos que criar um sobre o fenômeno da interferência ondular. Para isso, nós usamos papelão, papel, canetas e linha para criar um painel explicando os tipos de interferência com uma representação visual delas. Além disso, nós criamos um site, no qual eu participei na criação, usando HTML, CSS e Javascript para fazer uma representação mais detalhada dos tipos de interferência na prática. No final, acredito que conseguimos fazer a apresentação com clareza e ser criativos com a produção desse site.
